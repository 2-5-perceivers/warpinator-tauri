# [Warpinator Tauri](https://2-5-perceivers.github.io/#/warpinator)

A fast and secure Warpinator client built with Rust and Tauri.

**Note:** This project is currently in active development.

## Tech Stack

- **Framework:** Tauri (Rust)
- **Frontend:** React, TypeScript, npm
- **UI Components:** shadcn/ui

## Core Library

For protocol development and issue tracking regarding the underlying implementation, please refer to the core library repository:
[warpinator-lib](https://github.com/2-5-perceivers/warpinator-lib)

## Roadmap

Once the project reaches maturity, the goal is to distribute the application through the standard channels across Windows, macOS, and Linux.
